# SauceDemo Automation Framework

A scalable, production-grade test automation framework built with:
- **Java 17**
- **Selenium WebDriver 4** – UI automation
- **Cucumber 7 + Gherkin** – BDD test cases
- **TestNG 7** – Parallel test execution
- **REST Assured 5** – API testing
- **Allure Reports** – Test execution reports
- **WebDriverManager** – Auto browser driver management
- **Page Object Model (POM)** – Clean, maintainable structure

## Configuration
Edit `src/test/resources/config.properties` to change settings:

```properties
# Browser: chrome or firefox
browser=chrome

# App URL
base.url=https://www.saucedemo.com/

# API Base URL
api.base.url=https://simple-books-api.click
```
